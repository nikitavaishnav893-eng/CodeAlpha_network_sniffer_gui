# Basic Network Packet Sniffer (Python GUI)

This project is a basic network packet sniffer developed using **Python** and **Scapy**.
It captures live network traffic and displays packet details such as source IP,
destination IP, protocol, ports, and payload in a **GUI built with Tkinter**.
