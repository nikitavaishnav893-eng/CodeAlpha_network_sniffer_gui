#Basic Network Packet Sniffer (Python GUI)

This project is a basic network packet sniffer developed using **Python** and **Scapy**.
It captures live network traffic and displays packet details such as source IP,
destination IP, protocol, ports, and payload in a **GUI built with Tkinter**.



 Features
- Capture live network packets
- Display Source IP and Destination IP
- Identify Protocol (TCP / UDP)
- Show Source and Destination Ports
- Display packet payload
- Simple and user-friendly GUI



Technologies Used
- Python 3
- Scapy
- Tkinter (GUI)
- Kali Linux
 

 Requirements
- Kali Linux
- Python 3
- Scapy (`python3-scapy`)
- Tkinter (default with Python)


 How to Run
1. Open terminal in project folder
2. Run the program with root privileges:

sudo python3 packet_sniffer_gui.py
